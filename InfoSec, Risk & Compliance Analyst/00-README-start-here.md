## How to use each file
1. Read Part A–B (beginner/intermediate) fully; skim advanced parts on first pass.
2. Do every **self-test checklist** — all boxes must be YES before moving on.
3. Answer the **interview questions** at the end OUT LOUD. Reading ≠ interview readiness.
4. Return to advanced sections on second pass if time allows.

## If time collapses
Protect: **02 → 03 → 01(Part C) → 08**. Those four carry ~80% of the interview.

## The thesis to carry into the room
You operate technical controls AND you think in risk, evidence, ownership, and closure.
Every technical answer ends with the governance beat; every GRC answer grounds in a technical example.
That bridge is the job — and it's already how you work.
