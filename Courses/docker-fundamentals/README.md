# Docker Fundamentals

Learn the basics of Docker and how to containerize applications

## Who this course is for

This course is designed for developers and system administrators who want to learn how to containerize their applications and improve their deployment and management skills.

## What you'll be familiar with

- Understand the concept of containerization and its benefits
- Know how to install and configure Docker on a local machine
- Be able to create and manage Docker containers
- Understand how to use Dockerfiles to automate the build process
- Know how to use Docker Compose to manage multi-container applications
- Understand how to use Docker volumes and networks

## Prerequisites

- Basic understanding of Linux and command-line interfaces
